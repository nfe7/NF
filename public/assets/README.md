# Assets Directory

This directory is used to store static images for your portfolio.

## How to add your Profile Picture

1.  Take your profile photo.
2.  Rename it to exactly **`profile.jpg`**.
3.  Place it in this folder (`public/assets/`).
4.  The website is already configured to load it from here.

## How to add Project Cover Images (Optional)

If you want to reference other static images in your code:
1.  Place them here.
2.  Reference them in your code as `/assets/your-image-name.png`.
